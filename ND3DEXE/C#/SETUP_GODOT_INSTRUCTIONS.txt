================================================================================
INSTRUCCIONES PARA CONFIGURAR GODOT - WoldVirtual3D Viewer
================================================================================

Para que el visor 3D funcione, necesitas el ejecutable de Godot en una carpeta local.

OPCIÓN 1: Colocar Godot manualmente (RECOMENDADO)
---------------------------------------------------

1. Descarga Godot desde: https://godotengine.org/download
   - Versión: Godot 4.x (cualquier versión 4.0, 4.1, 4.2, 4.3, 4.4, o 4.5)
   - Plataforma: Windows 64-bit
   - Tipo: Standard (no headless)

2. Crea la carpeta Godot en:
   D:\woldvirtual3d\ND3DEXE\C#\bin\Release\net8.0-windows\Godot\

3. Copia el ejecutable de Godot a esa carpeta:
   - Puede llamarse: Godot.exe, Godot_v4.5-stable_win64.exe, etc.
   - Si tiene otro nombre, renómbralo a Godot.exe (opcional)

4. Estructura final:
   bin\Release\net8.0-windows\
   ├── WoldVirtual3D.Viewer.exe
   ├── Godot\
   │   └── Godot.exe (o Godot_v4.x-stable_win64.exe)
   └── [otros archivos...]

OPCIÓN 2: Usar godot_path.txt
------------------------------

Si prefieres colocar Godot en otra ubicación:

1. Crea un archivo llamado "godot_path.txt" en:
   D:\woldvirtual3d\godot_path.txt

2. Escribe la ruta completa a Godot.exe, por ejemplo:
   C:\Godot\Godot_v4.5-stable_win64.exe

3. Guarda el archivo y ejecuta el visor nuevamente

VERIFICACIÓN
------------

Después de colocar Godot, verifica:
- La carpeta Godot/ existe junto al ejecutable
- El archivo Godot.exe (o Godot_v4.x.exe) está dentro de la carpeta Godot/
- El archivo es ejecutable (no está bloqueado por Windows)

NOTAS IMPORTANTES
-----------------

- NO es necesario instalar Godot en el sistema Windows
- El ejecutable de Godot debe estar en la carpeta Godot/ junto al .exe del visor
- Funciona con cualquier versión de Godot 4.x
- El visor buscará automáticamente en múltiples ubicaciones
