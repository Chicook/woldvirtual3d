# Red P2P Distribuida

## Descripción

Directorio para almacenar configuración y datos relacionados con la red P2P distribuida.

## Estructura

- Configuración de nodos
- Datos de sincronización
- Logs de red
- Cache de peers

## Uso

La red P2P se inicializa automáticamente después del login exitoso mediante `P2PNetwork.cs`.
